# 城市位置选择Demo

[Download](https://github.com/shenhuanet/AndroidOpen/raw/master/--Downloads/Citypicker.zip)

基于本地json城市列表文件+高德定位api封装的城市位置选择demo.

## 调试时注意事项
- 注意Android6.0运行时权限开启
- 注意开发者需要在自己高德地图开放平台应用控制台中填写相应的值,并在app中配置
